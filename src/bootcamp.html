What is America's Promise Java Boot Camp?

America's Promise Java Bootcamp is an intensive bootcamp where you learn about Java web development. It goes from Monday to Friday, 9-5, and teaches you the basics of Java programming such as Object-Oriented Programming, as well as basic HTML and CSS so you can develop a fully functioning website from scratch. The course is tough, and there's a lot of work, but you learn a lot as well. Everything I've learned so far has come from taking this boot camp. If you want to learn more, check out this website.
