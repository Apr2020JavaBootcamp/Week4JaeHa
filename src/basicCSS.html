Basics of CSS and Bootstrap

5/14/2020

After some HTML we started talking about CSS and Bootstrap. CSS is another important component of web development, which is changing how your web page looks so it is unique and visually interesting. Without CSS, all our web pages would look very bland. On top of that, CSS is important for organizing and presenting information so users can read it easily.


In our boot camp, we only had 8 week to learn how to build a complete website, so we didn't have much time to learn much about HTML and CSS. However, Bootstrap is a great tool for creating good-looking and mobile-friendly websites without having to know a lot about CSS or HTML.


What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files
  • CSS files can be written with the same editors your HTML editors

CSS Syntax

A CSS rule-set consists of a selector and a declaration block:



  • The selector points to the HTML element you want to style.
  • The declaration block contains one or more declarations separated by semicolons
  • Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

What is Bootstrap?

Bootstrap is a free and open-source framework for CSS designed for responsive mobile-first web development. Bootstrap has a collection of design templates that users can utilize in order to make their web pages look presentable without having to learn a lot of CSS or HTML. Bootstrap lets you use tools like cards, navigation bars, and more to make your websites look better with less effort. On top of that, websites made with Bootstrap are more mobile-friendly.
